YAML::PP - Preserve

  • Quoting style can be preserved (still experimental)
  • Scalars will be loaded as an object with overloading
use YAML::PP::Common qw/ PRESERVE_SCALAR_STYLE /; my $yp = YAML::PP->new( preserve => PRESERVE_SCALAR_STYLE ); say $yp->dump_string( $yp->load_string( $yaml ) );
