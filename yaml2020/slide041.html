YAML::PP - Preserve

  • Hash key order can be preserved when loading and dumping
  • Implemented with tie()
use YAML::PP::Common qw/ PRESERVE_ORDER /; my $yp = YAML::PP->new( preserve => PRESERVE_ORDER ); say $yp->dump_string( $yp->load_string( $yaml ) );
