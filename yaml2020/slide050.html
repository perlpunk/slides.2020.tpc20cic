YAML::PP - Include options

  • Allows nested includes, but not cyclic
  • Searches relative to current YAML file
my $include = YAML::PP::Schema::Include->new( # allow !include /etc/passwd # or !include ../../../../etc/passwd allow_absolute => 1, ); my $include = YAML::PP::Schema::Include->new( paths => [ qw( /path/prefix ) ], );
